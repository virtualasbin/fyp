Aim (50 words): 
The aim of this project is to design and develop a secure Progressive Web Application (PWA) 
that provides students with an installable, cross-device task management system. The system 
will incorporate push notifications, offline functionality, smart reminders, and productivity 
analytics to enhance time management and academic performance. 
Problem description (100 words): 
Many students struggle to manage multiple assignments, coursework deadlines, and daily 
academic responsibilities (Claessens, 2007). Traditional tools, such as paper planners or generic 
note-taking apps, often lack proactive reminders, task prioritisation, and cross-device 
accessibility, leading to missed deadlines and increased stress. Existing task management 
solutions may be overly complex, subscription-based, or not tailored to academic needs. There 
is a clear need for a modern, web-based system that provides app-like functionality, including 
offline access, background notifications, and mobile installability, allowing students to efficiently 
organise, track, and prioritise tasks regardless of device or location, thereby improving 
productivity and academic performance. 
Intended outcomes (100 words):  
The project will deliver a fully functional Progressive Web Application (PWA) that can be 
installed on both mobile and desktop devices, providing students with an app-like experience 
without requiring app store distribution. The system will enable secure user registration and 
authentication, task creation and management, deadline tracking, push notifications, offline 
functionality, and structured task data storage in a relational database. Additional features will 
include calendar integration, productivity analytics, and performance dashboards. 
Comprehensive testing and usability evaluation will ensure reliability, accessibility, and security. 
Final deliverables will include the deployed application, technical documentation, testing 
evidence, user feedback, and a reflective project report.
